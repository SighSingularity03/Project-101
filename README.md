# Project 101

Portfolio exercise demonstrating development environment setup, AI-assisted tooling, and version control with GitHub.

## Tools Installed

- **Cursor** — primary IDE
- **Claude Code** (extension) — AI coding assistant
- **Codex** (extension) — AI coding support

## Work Completed

1. Installed Cursor and configured the Claude Code extension
2. Added the Codex extension via the terminal
3. Created a GitHub account and a public repository
4. Connected the repository to Cursor
5. Cloned the repo and updated this README
6. Pushed an initial test commit
7. Revised the documentation and pushed a follow-up commit

## Challenges & Solutions

**GitHub ↔ Cursor integration**  
Resolved by following GitHub documentation and consulting AI assistants (Claude, ChatGPT).

**SSH key authentication**  
Generated an SSH key pair in the terminal and added the public key to GitHub.

**Pushing changes to remote**  
Used the standard Git workflow: `git add` → `git commit` → `git push`.
